# R-Converter PRO

**Convertitore Broadcast per LED Wall - Immagini e Video Multi-Layer**

> Versione 2.0.0 PRO | Windows 10/11

---

## Quick Start

```bash
pip install -r requirements.txt
python main.py
```

**Portable**: Doppio click su `R-Converter_Portable.exe` (anche da USB).

**Build**: Doppio click su `_clean_and_build.bat`.

## Caratteristiche Principali

- Multi-layer collage con drag & drop (windnd)
- 6 preset LED wall, 5 software target (Resolume, vMix, Millumin, H.264, H.265)
- Export broadcast: DNxHR, HAP Q, ProRes, pipeline ottimizzata
- Tema dark, pannello PRO con dropdown a cascata
